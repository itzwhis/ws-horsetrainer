# WS Horse Trainer

A complete and immersive Horse Trainer job system for RedM servers.

Train horses, complete jump training activities, improve horse stats, and build stronger bonds with your horses through realistic western gameplay mechanics.

Built for immersive roleplay servers with modern UI and optimized performance.

---

## Features

- Horse training system
- Jump training activities
- Horse progression mechanics
- Bond & trust system
- Reward system
- Stable interactions
- Modern immersive UI
- Multiplayer compatible
- Optimized performance

---

## Horse Stats

Players can improve:

- Speed
- Stamina
- Health
- Bond Level
- Handling

---

## Included UI

- Horse Information Menu
- Horse Training Interface
- Horse Stats Panel
- Progress UI
- Reward UI
